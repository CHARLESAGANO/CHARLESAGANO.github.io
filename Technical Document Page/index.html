Introduction to CSS

CSS stands for Cascading Style Sheets. CSS is responsible for the look and feel of a webpage built with HTML. CSS makes a webpage build with HTML visually appealing.

I have learned the basics of CSS from this course. The topic that I have learned are

  • Different types of CSS: Inline CSS, Internal CSS and External CSS
  • Syntax of CSS: The Selector, Property and Value makes up a basic CSS
    1. Selector: are used to find the HTML element you want to style
    2. Property: determines an HTML elemt's style or behavior
    3. Value: the exact characteristic applied to a property of an HTML element.
  • Commonly used CSS properties:
    1. Text Syles: Properties like color,font-family,font-size,front-weight and text-declaration control the appearance of text.
    2. Background: Properties like background-color, background-image, and background-position allow you to set background colors or images for your webpage elements.
    3. Borders: Properties like border-style, border-width and border-color define the style, thickness and color of border around elements.
    4. Margind and Padding: margin sets the space around an element outside its border, while padding sets the space between the element's border and its content.
    5. Flex Box: a powerfule CSS layout model for arranging elements on a webpage. It provides a simple and more intuitive way to control how elements resize and position themselves within a container.
